#spoofing 
Es una practica ilegal que consiste en sumplantar la identificacion de una persona para oculatr la suya propia

### Port security

es el metodo y efectuvi para prevenir los ataques de desbordamiento de la tabla de dirección
MAC es habilitar port segurity "seguridad imediata"

### La configuración

- manual se usa el skitin 
  protect -> solo genera conteo de cuantos esta entrando al servidor
  restrict -> no apaga el puerto y se manda un  log al servidor entonces alerta que estan bajo ataque
  shutdow -> este biene por defecto y apaga el LED del puerto mas bien cierra las compuertas
